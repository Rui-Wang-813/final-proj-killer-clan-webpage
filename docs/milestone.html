Final Project Milestone Report: Font Rendering

Rui Wang, Zhuowen Chen, Yifan Zhong, Yicheng Sun



Brief summary

In this project, we are going to build a font rendering engine which is enable to render a sequence of characters from a given .ttf font file. In this milestone, we use FreeType and SVG resource online and currently support rendering multiple characters. We are still planning to achieve font rasterization in this project in our final deliverables.

Preliminary Results

First, we used the external library Free Type to load the .ttf files. Then, we take out the read data and use this data to render characters of given font. We initially implemented the function to write svg files for a single character. Then we used this function to further implement the function that can write svg files for multiple characters in a line.
